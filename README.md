Логгер ошибок в mysql DB
===================================

Информация о модуле
-------------------

Модуль предоставляет возможность добавлять отзывы к элементам.

Установка
------------

1) Добавить в `composer.json` файл проекта.

```
"skeeks/cms-log-db-target": "*"
```

2) Запуск миграций и необходимых установок.

```
php yii cms/update
```
